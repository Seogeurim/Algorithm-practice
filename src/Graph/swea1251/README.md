### 크루스칼 알고리즘 vs 프림 알고리즘

이 문제는 모든 노드를 연결하는 최소 비용을 찾는 문제이다. 특히 특정한 간선이 주어지지 않았기 때문에 모든 노드끼리의 간선의 비용을 다 구해야 한다. 이를 생각하면 알고리즘의 특성을 고려했을 때 크루스칼 알고리즘보다 **프림 알고리즘**이 더 효율적이다. 문제에 따라서 더 효율적인 알고리즘을 생각해서 풀면 좋을 것 같다.
